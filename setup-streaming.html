🎮 Setup Streaming

Configura le tue API per iniziare a streammare!

💜

Twitch

▼
1 Vai su 🔗 Twitch Developer Console
2 Clicca "Register Your Application"
3 Nome: Stardew Live Stream
4 OAuth Redirect URL: https://tuodominio.com/auth/twitch
5 Categoria: Game Integration
🎵

TikTok

▼
1 Vai su 🔗 TikTok for Developers
2 Crea "New App"
3 Nome: Stardew Live Stream
4 Redirect URI: https://tuodominio.com/auth/tiktok
5 Scopes: user.info.basic, live.room.info
💡 Le tue chiavi API vengono salvate solo nel tuo browser (localStorage)
🔒 Nessun dato viene inviato a server esterni
